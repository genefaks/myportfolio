Augmented Reality Application


AR Design / Unity / Vuforia

AR mobile application / About the Project

I created an Augmented Reality application where I used an android device to move around but when recognized by a certain object an overlay video is seen on that object.

How it was accomplished

The Augmented Reality application was created using Unity 2019.1.14f. Other resources include the Vuforia augmented reality support and Vuforia developer portal, used in this project. Vuforia is used for creating augmented reality apps for ios and android devices. Vuforia developer portal was used for creating the license and database required for the app.

Resources used in creating the AR application project:



Unity 2019.1.14f


Vuforia augmented reality support


Android build support


Vuforia developer portal

Reflection

Personally I want to explore more about Augmented Reality apps spending good hours experimenting and creating projects to gain better knowledge, confidence and usability of it. Tutorials from Dr. Simon Colreavy-Donnelly was helpful and useful during the whole process and creation of my project. Working on this project again, I would make use of animations in the scene like running, sitting, walking.
